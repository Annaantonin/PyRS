To start main window from analysis machine

.. code-block::

   $ PYTHONPATH=$PWD:$PYTHONPATH QT_API=pyqt python scripts/pyrsplot

To develop

.. code-block::

   $ pyrsdev.sh
   
To run the tests


.. code-block::

   $ pyrstest.sh
